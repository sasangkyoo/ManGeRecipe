  **요리 초보자를 위한 추천 레시피 시스템**을 구축하고자 합니다.

## 5. ⚙️ 주요 기능

- **페이지 기반 레시피 리스트 페이지(1~25페이지) 크롤링**
  - 목록에서 요약 정보(제목, 작성자, 조회수, 이미지, 링크) 수집
- **각 레시피 상세 페이지 접근 및 세부정보 수집**
  - 인분 수, 조리 시간, 난이도, 재료/양념 텍스트 구성
- **CSS Selector 기반 정밀 파싱**
  - 예외 케이스 대비한 데이터 누락 보완 로직 포함
- **수집 결과 CSV로 자동 저장**
  - `food_data.csv`, `recipe_data.csv`로 각각 저장

---

## 6. 💡 데이터 활용 계획

- 난이도, 시간 기반 레시피 카테고리 분류
- 재료/양념 조합 기반 추천 알고리즘 개발
- 요리 초보자/1인 가구용 맞춤형 레시피 추천 서비스 기획
- 식재료 트렌드 분석 및 마트 연계형 추천 시스템 구축

---

## 7. 🧯 예외 처리

크롤링 중 발생할 수 있는 네트워크/HTTP 오류를 다음과 같이 처리합니다:

| 예외 | 설명 |
|------|------|
| `HTTPError` | 서버 응답 오류 (4xx, 5xx 등) |
| `ConnectionError` | 네트워크 연결 실패 |
| `Timeout` | 요청 시간 초과 |
| `RequestException` | 기타 요청 오류 |
| `Exception` | 기타 파싱 또는 예외적 오류 |
